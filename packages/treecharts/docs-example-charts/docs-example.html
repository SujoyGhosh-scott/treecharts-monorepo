TreeCharts Renderer Examples

Explore different connection styles with a simple tree structure

Direct Renderer

Clean, straight lines connecting parent and child nodes

Best for: Simple hierarchies, minimal visual complexity, clear parent-child relationships

Right Angle Renderer

Orthogonal connections with clean 90-degree angles

Best for: Organizational charts, flowcharts, structured diagrams with clear alignment

Curved Renderer

Smooth, curved connections for an organic appearance

Best for: Natural hierarchies, family trees, mind maps, artistic presentations

All Direction Renderer

Radial layout with nodes distributed in all directions

Best for: Network diagrams, radial trees, space-efficient layouts, complex relationships

All examples use the same simple tree structure: Root node A → Children B & C → B has children D & E
