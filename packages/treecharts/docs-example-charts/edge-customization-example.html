Edge Customization

Comprehensive edge styling demonstration showing how to customize connection lines between tree nodes. This example showcases dashed lines, custom colors, arrow indicators, and text labels on edges for enhanced visual communication.

Dashed Lines Use dasharray: "5,5" for dashed edge patterns
Custom Colors Independent colors for edges, arrows, and text labels
Arrow Indicators Directional arrows showing parent-child relationships
Edge Text Labels Descriptive text on connections with background styling
Line Thickness Configurable line width and opacity settings
Right-Angle Layout Professional L-shaped connections for organizational charts
Key Configuration:
• edgeConfig.color: "#e74c3c" (red edges)
• edgeConfig.dasharray: "5,5" (dashed pattern)
• edgeConfig.showArrows: true (arrow indicators)
• edgeConfig.textColor: "#8e44ad" (purple text)
• edgeText: "label" (text on each connection)
