TreeCharts Alignment Examples

Horizontal Alignment (verticalAlign)

The verticalAlign property controls the horizontal positioning of nodes within each level. Available options: "left", "center", "right". Default is "center".

Center Alignment

Vertical Alignment (horizontalAlign)

The horizontalAlign property controls the vertical flow direction of the tree. Available options: "top-to-bottom", "bottom-to-top". Default is "top-to-bottom".

You can combine both horizontal and vertical alignment options for different layouts. These examples show simple color variations to distinguish the different alignment combinations.
